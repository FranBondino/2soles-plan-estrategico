# Plan Estratégico Dos Soles: Evolución de Marca & Ecosistema Digital

Este repositorio contiene el material analítico y de presentación del **Plan Estratégico de Marketing, Comunicación y Posicionamiento** para la marca **Dos Soles** (Estrategia de crecimiento híbrida B2B + B2C para los próximos 12 meses).

## Contenido del Repositorio

1. **`analisis_plan_dos_soles.md`**: Documento con el análisis estructurado de las 55 páginas del diagnóstico original. Incluye:
   - Diagnóstico estratégico y la brecha de percepción (Distribuidor vs. Referente).
   - Matriz FODA detallada de la empresa.
   - Definición del Ecosistema Omnicanal y el embudo orgánico.
   - Segmentación de audiencias (Customer Personas) y prioridades de implementación.

2. **`presentacion_dos_soles.html`**: Presentación ejecutiva e interactiva de diapositivas (13 slides). 
   - **Branding Adaptado:** Diseñada con la tipografía oficial *Plus Jakarta Sans*, color fucsia/magenta corporativo (`#A7144C`) y acentos solares, con iconos SVG limpios y responsive.
   - **Navegación Avanzada:** Flechas, barra espaciadora, scroll del mouse y controles táctiles para dispositivos móviles.
   - **Edición en Vivo:** Presionando la tecla **`E`** (o pasando el mouse por la esquina superior izquierda) se habilita el modo de edición de texto directo sobre las diapositivas con guardado automático y opción para exportar los cambios.
